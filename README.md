# Club Class Booking System

Sistema de reservas de clases personalizadas para clubes deportivos.

Este proyecto permite a los socios reservar clases con entrenadores utilizando un formulario web, un flujo de automatización en n8n y una API en FastAPI conectada a Supabase.

## Features

- Reserva de clases personalizadas
- Validación automática de socios
- Prevención de conflictos de horario
- Procesamiento de lenguaje natural para horarios
- Integración con Supabase
- Automatización con n8n
- Registro de reservas en Google Sheets
- Preparado para integración con Google Calendar

## Arquitectura

Frontend → n8n → API → Supabase → Reportes
